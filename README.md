# HappyRoots 🌿

A personal Hoya plant care scheduler with adaptive watering schedules, Google Drive sync, and an AI care assistant.

## Setup

1. Deploy to Netlify (connect this GitHub repo)
2. After deploy, go to your Google Cloud Console:
   - APIs & Services → Credentials → Your OAuth Client
   - Add your Netlify URL to **Authorized JavaScript origins** (e.g. `https://happyroots.netlify.app`)
3. Open your app URL, tap **Connect Drive**, sign in with Google
4. Your data syncs automatically across all devices

## Development

```bash
npm install
npm run dev
```
